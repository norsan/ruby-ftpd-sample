ruby-ftpd-sample
================

ftp daemon sample by ruby.

available ftp commands are:
  QUIT
  USER
  PASS
  ABOR
  CDUP
  CWD
  PWD
  TYPE
  PORT
  PASV
  LIST
  RETR
  STOR

This repository contains follow files:
  ftpd.rb: ftp daemon program.
  ftpd.yml: setting file.
  README.md: this file.
  LICENSE: license information(MIT License).
